Important FAQ's

What is the Document Object Model(DOM)?

The Document Object Model specifies how browsers should create a model of an html and how javascript can access and update or edit the webpage within a browser window

How Do you select an element from the DOM?

by using document.getElementById("") for Id,document.getElementByClass("") for classes,document.getElementByTagName("") for tags,document.querySelector("") for selecting hte first query,document.querySelectorAll("") for selecting all elements that matches the query

What is the Event Delegation in the context of the DOM,and why is it useful?

In this case,you attach an .addEventListener() to an parent element so that you can control a child element's action.It is efficient in case of large number of elements are there to handle ,the large number of elements can be handled by it's parent element's command

How do you manipulate an element's attributes and styling using the DOM?

To get attribute use element.getAttribute(attributeName) & To set attribute use element.setAttribute(attributeName,newValue).
For styling,use: element.style.propertyName = "newValue" to style a part of html through javascript language
